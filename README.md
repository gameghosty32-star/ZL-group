# ZL-group

Uma página web moderna e responsiva para apresentar a ZL-group, sua empresa, serviços, galeria e parceiros.

## 📋 Descrição

ZL-group é um site corporativo que apresenta informações sobre a empresa, seus serviços, empresas parceiras e galeria de projetos. O site foi desenvolvido com HTML, CSS e JavaScript para proporcionar uma experiência de usuário fluida e intuitiva.

## 🎯 Funcionalidades

- **Página Inicial** - Apresentação principal da empresa
- **Serviços** - Listagem detalhada dos serviços oferecidos
- **Empresas Parceiras** - Showcase das empresas com as quais trabalhamos
- **Galeria** - Portfolio visual de projetos e trabalhos realizados
- **Responsivo** - Design adaptável para todos os dispositivos (desktop, tablet, mobile)
- **Interativo** - Elementos dinâmicos com JavaScript

## 📁 Estrutura do Projeto

```
ZL-group/
├── index.html          # Página principal
├── servicos.html       # Página de serviços
├── empresas.html       # Página de empresas parceiras
├── galeria.html        # Página de galeria
├── css/                # Arquivos de estilo
├── js/                 # Arquivos JavaScript
├── img/                # Imagens do projeto
├── videos/             # Vídeos utilizados no site
└── empresas/           # Recursos relacionados a empresas
```

## 🛠️ Tecnologias Utilizadas

- **HTML** (55.5%) - Estrutura das páginas
- **CSS** (40.8%) - Estilização e responsividade
- **JavaScript** (3.7%) - Funcionalidades interativas

## 🚀 Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/gameghosty32-star/ZL-group.git
   ```

2. Navegue até a pasta do projeto:
   ```bash
   cd ZL-group
   ```

3. Abra o arquivo `index.html` no seu navegador preferido:
   ```bash
   open index.html
   ```

   Ou, se preferir usar um servidor local (recomendado):
   ```bash
   python -m http.server 8000
   ```
   Então acesse `http://localhost:8000` no seu navegador.

## 🎨 Páginas

### index.html
Página inicial com apresentação da empresa, hero section e destaques dos serviços.

### servicos.html
Página dedicada aos serviços oferecidos pela ZL-group com descrições detalhadas.

### empresas.html
Seção que apresenta as empresas parceiras e colaboradores da ZL-group.

### galeria.html
Galeria de imagens e projetos realizados pela empresa.

## 📝 Notas

- Certifique-se de que todas as imagens estão na pasta `img/`
- Os vídeos devem estar organizados na pasta `videos/`
- Para melhor performance, comprima as imagens antes de fazer upload
